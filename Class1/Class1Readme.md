# 🎮 Python Mini Projects Collection

A simple collection of beginner-friendly Python programs to practice core programming concepts like conditionals, loops, and user input.

---

## 🚀 Features

This project includes 3 small programs:

### 1️⃣ Odd or Even Checker

* Takes a number as input
* Determines whether the number is **odd** or **even**

---

### 2️⃣ Grade Calculator

* Asks the user for a score (0–100)
* Outputs a grade:

  * **A** (90–100)
  * **B** (80–89)
  * **C** (70–79)
  * **D** (60–69)
  * **E** (<60)
* Handles invalid inputs (above 100)

---

### 3️⃣ Number Guessing Game 🎯

* Generates a random number between **1 and 100**
* User keeps guessing until correct
* Provides feedback:

  * “Too high”
  * “Too low”
* Tracks number of attempts

---

## 🧠 Purpose

This project is designed for beginners to:

* Practice **if-else conditions**
* Learn **loops (while)**
* Understand **user input/output**
* Use the **random module**

---
